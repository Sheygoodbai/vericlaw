# VeriClaw 爪印 Privacy

This repository is a GitHub launch surface for VeriClaw 爪印.

Repository-side processing is limited to ordinary GitHub usage such as issues,
pull requests, stars, forks, and repository browsing.

Product-side behavior may differ from repository-side behavior. Public launch
copy in this repository describes the intended product direction, but the
product runtime and device behavior are separate from GitHub itself.
