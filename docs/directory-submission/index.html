Submission Form Answers

These blocks are meant for common fields on plugin or tool directory forms.

What does it do?

VeriClaw 爪印 is the Apple-native correction companion for OpenClaw. It is designed for cases where an AI agent hallucinates, fake-completes, drifts from role, or needs evidence-first verification before calling work done.

Who is it for?

Teams, builders, and operators working with OpenClaw or similar AI agent workflows who need correction-first supervision rather than passive monitoring alone.

How do users install it?

Use clawhub install vericlaw for the shortest discovery route. Inside OpenClaw, use openclaw skills install vericlaw or openclaw plugins install clawhub:vericlaw.

How should it be differentiated?

Position VeriClaw around correction, diagnosis, intervention, and verification, not as a generic monitoring dashboard or a standalone public MCP server.
