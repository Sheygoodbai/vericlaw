What “AI work hallucination” means here

The model does not merely hallucinate facts. It hallucinates work completion: claiming a fix shipped, a task ran, a file changed, a request succeeded, or a review happened when the evidence trail is empty, partial, or inconsistent.

  • The agent says it fixed the bug, but the output or code stayed the same.
  • The bot says it checked something, but there is no artifact or proof.
  • The workflow closes as “done” before anyone verifies the actual state.
  • The narration sounds complete, but the execution was skipped or only partial.
