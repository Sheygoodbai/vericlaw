这些词通常在找什么

这类搜索几乎都已经到了“要不要继续信这个 agent”的阶段，转化强度很高， 也是最适合直接引向安装和核验闭环的时刻。

未做却说做了

执行没有发生，但报告、状态或口头反馈已经把任务宣称为完成。

明明没做却说做了

用户用更自然的口语在描述同一个失败：模型把设想、计划或空转当成完成。

装作做完了

这是典型的假完成问题，后续重点是补证据、纠偏和核验，而不是继续追问。
