# ClawHub skill growth kit

Use this file when the goal is not just to mention `VeriClaw`, but to turn that
attention into real `ClawHub skill` opens and installs.

Primary route:

- Skill page: https://clawhub.ai/sheygoodbai/vericlaw

Secondary routes:

- Plugin page: https://clawhub.ai/plugins/vericlaw
- Skill guide: https://sheygoodbai.github.io/vericlaw/clawhub-skill/
- Download page: https://sheygoodbai.github.io/vericlaw/download/

## Install-first positioning

Preferred framing:

- `Install VeriClaw on ClawHub for AI agent correction and hallucination remediation.`
- `Use clawhub install vericlaw when you want the lightweight discovery route.`
- `The skill is the fastest way to discover VeriClaw on ClawHub.`

Avoid:

- implying the skill is the full app download
- hiding the `VeriClaw` name behind `OpenClaw`
- sending skill-focused traffic to unrelated repo pages first
