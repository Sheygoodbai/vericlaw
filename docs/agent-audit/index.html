What agent audit usually needs

Directory sites and reviewers often collapse several problems into one bucket. This page makes the intent explicit and points those visitors back to the canonical VeriClaw surfaces instead of scattering them across generic descriptions.

Audit the claim

When an agent says the work is finished, the first step is checking whether the completion is real, partial, or invented.

Audit the evidence

Correction becomes tractable when the operator can compare claimed actions with observable files, outputs, and tool traces.

Audit the recovery

The operational goal is not just catching drift. It is bringing the workflow back to a truthful, verifiable state.
