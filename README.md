# VeriClaw 爪印

![VeriClaw mark](docs/assets/vericlaw-lobster-mark.svg)

`VeriClaw 爪印` is an Apple-native correction companion for OpenClaw.

This repository is the public GitHub launch surface for the VeriClaw story:

- product positioning
- launch copy
- legal and redistribution materials
- market differentiation notes
- first-use UX documentation

## What VeriClaw does

VeriClaw is built for the moment when a bot drifts, hallucinates, overreaches,
or stops honoring its professional role.

The correction loop is:

- evidence
- diagnosis
- prescription
- verification
- casebook learning

## Product stance

- `Companion, not clone`
  VeriClaw strengthens OpenClaw rather than pretending to replace it.
- `Correction, not just monitoring`
  The goal is to answer what should happen next, not only what went wrong.
- `Professional-role drift`
  Many failures are framed as occupational-role contract failures when useful.
- `Apple-native supervision`
  The intended experience is a coherent native macOS and iPhone supervision flow.

## Current release posture

- GitHub is the current public launch surface
- Apple companion hardening continues in parallel
- watchOS is outside the current release gate

## Repository scope

This repository currently focuses on launch materials and public documentation
for VeriClaw 爪印.

OpenClaw remains the upstream runtime and gateway substrate. Where this
repository discusses OpenClaw, that use is referential and compatibility-based.
